# 🎮 Pixel Game

Game sederhana berbasis web dengan gaya **pixel art retro**, dikembangkan menggunakan **HTML, CSS, dan JavaScript**.  
Game ini dirancang sebagai proyek latihan untuk memahami **animasi sprite**, **interaksi pemain (player control)**, dan **mekanika NPC (non-player character)**.

---

## 🌟 Fitur Utama
- 🎨 **Desain Pixel Art:** tampilan visual bergaya klasik yang ringan dan nostalgic.  
- 🧍‍♂️ **Karakter Player & NPC:** pemain dapat bergerak, berinteraksi, dan menjelajah area game.  
- ⚙️ **Sistem Gerak NPC:** musuh atau karakter lain memiliki area patroli otomatis.  
- 🕹️ **Kontrol Keyboard:** menggunakan tombol arah untuk bergerak.  
- 💾 **Berbasis Web:** dapat dimainkan langsung di browser tanpa instalasi.

---

## 🧠 Teknologi yang Digunakan
- **HTML5 Canvas** untuk tampilan visual dan render game.  
- **CSS3** untuk layout dasar dan efek visual tambahan.  
- **JavaScript (Vanilla)** untuk logika permainan, kontrol pemain, dan animasi sprite.
